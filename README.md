##  Features

-  Track daily shifts and working hours
-  Record tips and calculate total earnings
-  Automatically generate bi-weekly payroll summaries
-  Clear and simple UI focused on usability
-  Cloud-based data storage with Supabase

---

##  Tech Stack

- **Frontend**: Next.js, React
- **Styling**: Tailwind CSS
- **Backend / Database**: Supabase
- **Deployment**: Vercel

---

## Why This Project?

This project was built for personal use, not as a tutorial exercise.

While working at a restaurant, I needed a simple way to:

Track shifts accurately

Calculate tips and total pay

Prepare reports I actually submit to my manager

Building SushiTime helped me practice real-world problem solving, data handling, and frontend development using modern tools.


## Future Improvements
